CSS structure for Generic Outdoor Theme

- `scss/` — source SCSS files
  - `base/` — variables, mixins, reset, base styles
  - `components/` — buttons, forms, etc.
  - `layout/` — header, footer, grid
  - `utils/` — utility classes
- `dist/` — compiled CSS output (gitignored by default)

Build: run your bundler to compile `scss/main.scss` into `dist/main.css`.
